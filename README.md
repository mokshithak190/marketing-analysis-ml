# Marketing Analysis ML Project

This project analyzes marketing and product performance data using Python and pandas.

## Files Included:
- marketing_analysis.ipynb (Jupyter Notebook)
- marketing_data.csv (Dataset)

## Steps Performed:
- Data loading
- Data exploration (EDA)
- Statistical summary
- Correlation analysis

## Dataset Source:
https://www.kaggle.com/datasets/imranalishahh/marketing-and-product-performance-dataset

## How to Run:
Open the notebook in Jupyter and run all cells.

## Objective:
To analyze how marketing factors affect revenue and customer behavior.# marketing-analysis-ml
